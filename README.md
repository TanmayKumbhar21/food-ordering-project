# 🍔 Food Ordering System

A Full Stack Food Ordering Web Application developed using Java Servlets, HTML, CSS, JavaScript, and MySQL. This project allows users to browse food items, register, log in, place orders, and manage their food ordering experience.

---

## 🚀 Features

- User Registration
- User Login Authentication
- Browse Food Menu
- Add to Cart
- Place Order
- Cash on Delivery
- Responsive User Interface

---

## ⚙️ How to Run

1. Clone the repository
2. Import the project into NetBeans.
3. Configure MySQL Database.
4. Import the SQL file.
5. Configure database credentials.
6. Run the project on Apache Tomcat.
